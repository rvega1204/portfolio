Follow these steps to set up the project locally.

### Prerequisites

- Node.js (v18 or higher recommended)

### Installation

1.  **Clone the repository:**

    ```bash

    cd portfolio
    ```

2.  **Install dependencies:**

    ```bash
    npm install
    ```

3.  **Run the development server:**

